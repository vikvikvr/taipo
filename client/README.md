# Taipo client

Solo project for Codeworks!

## Requirements

- Countdown before game starts ✅
- Multiplayer: 1 vs 1 ✅
- Room code ✅
- See opponent progress ✅
- Visualize opponent mistakes ✅
- Animations for mistakes ✅
- Sounds for events ✅
- Login with Google ✅
- Login with GitHub
- Login with Facebook
- Game over screen ✅
